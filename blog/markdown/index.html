Markdown 語法教學！

By 小蘇 13 May 2021

Markdown語法練習：

1.標題(#)：共6級

我是4級標題

2.內文：

換行 ==> 需要 空白列

3.粗體(** **)：

我是粗體

4.斜體(* *)：

我是斜體

5.有序序列(1. )：

  1. 項次1(有序序列)

    • 我是縮排

  2. 項次2(有序序列)

6.無序序列(* )：

  • 無序序列1

    • 我是縮排

  • 無序序列2

7.引用(> )：

我是 “引用”

8.超連結([]中括號放呈現文字()小括號放網址)：

我是超連結到我的github

第1篇文章超連結

9.圖片(！［］（）只要在小括號放網址)：

10.單行程式碼( ‘程式碼’ )：使用鍵盤左上角的 引號`

print("Hello World!") 這是python輸出指令

11.多行程式碼( ‘‘‘程式碼’’’ )：使用鍵盤左上角的 引號`

print("我是小蘇")
print("我是多行程式碼")

print("指定為python語言呈現樣式")
print("我是多行程式碼")

var s = "我是JavaScript";
alert(s)

12.表格 (對表格支援不好，無法使用分割或合併儲存格)：

  • 預設：靠左對齊
  • :—-: 置中對齊
  • —-: 置右對齊
例如：
| 欄1 預設 靠左   | 欄2 置中       | 欄3 靠右 |
| ------------- |:-------------:| -------:|
Tables 欄2 置中 欄3 靠右 備考
表格預設靠左 靠左對齊 $100
欄 3 靠右對齊 $1600
欄 2 置中對齊 $12
使用 ：—： 罝中對齊 $1
使用 —: 靠右對齊 $100
